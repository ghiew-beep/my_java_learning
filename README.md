# Java Learning Master Repository

## Overview

This repository contains structured Java learning modules covering core programming concepts, object-oriented programming, collections, file I/O, multithreading, JAR packaging, JDBC, and unit testing.

Each module is organized independently and follows incremental difficulty progression.

- Basic data structures
- Control flow
- Core syntax exercises

### module01_oop_collections
Object-Oriented Programming and Collections Framework:
- Classes and objects
- Inheritance and polymorphism
- Encapsulation
- Java Collections (List, Set, Map)
- Iterators and generics

### module02_io_files
File handling and input/output operations:
- File reading/writing

### module03_threads
Multithreading and concurrency:
